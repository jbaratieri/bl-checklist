Manual do Checklist Interativo

Baratieri Luthieria — 2025-08-23

1. Propósito

Este manual descreve como utilizar o checklist interativo para a construção artesanal de instrumentos de corda, com foco em fluxo de trabalho, registro de imagens e organização por projeto.

2. Estrutura do Projeto

  • Landing page: apresentação e acesso ao checklist.
  • Checklist: etapas, medidas e anexos por subetapa.
  • Assets: logotipos e ícones.

3. Imagens e Persistência

As imagens são comprimidas automaticamente (JPEG, até 1600×1200 @ ~80%) antes de serem salvas, garantindo fluidez e economia de espaço.

No primeiro carregamento, imagens antigas são migradas e recomprimidas para manter o armazenamento otimizado.

4. Organização por Projeto

Mantenha um identificador de projeto/instrumento ativo para separar as imagens e o progresso de cada instrumento construído.

5. Boas Práticas

  • Rode o app via HTTP (ex.: VS Code Live Server).
  • Evite fotografias de altíssima resolução; 1600×1200 é suficiente para registro.
  • Use descrições claras nas subetapas e mantenha o histórico fotográfico.
